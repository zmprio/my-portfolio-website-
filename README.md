# Md Asibuzzaman Prio - Portfolio

A modern, minimal, and fully static personal portfolio website designed for an Electrical & Electronic Engineering (EEE) student specializing in Telecommunications & Signal Processing.

## 🚀 Features
- **Clean & Professional Design**: Tailored for an engineering focus.
- **Fully Responsive**: Adapts seamlessly from desktop monitors down to mobile screens using CSS Grid/Flexbox.
- **Dark Theme**: Eye-catching dark navy aesthetics with a neon teal accent (`#64ffda`).
- **Interactive**: Smooth scrolling, sleek glassmorphism navigation, and elegant image hover states.
- **Completely Static**: Zero dependencies, no heavy frameworks, and requires absolutely no backend. Lightning fast load times.

## 🛠️ Technologies Used
- **HTML5**: Semantic tags used to ensure an accessible layout structure.
- **CSS3**: Custom properties (CSS variables) for robust theming without needing Tailwind or Bootstrap.
- **Vanilla JavaScript**: Lightweight DOM manipulation for the mobile hamburger menu and scroll effects.
- **Google Fonts**: Uses the 'Inter' typeface for clean, readable typography.
- **Font Awesome**: Used for crisp, scalable vector icons.

## 💻 How to Run Locally
Because this project is entirely static, there is no build process or package installation required.
1. Clone or download the repository.
2. Open the `index.html` file in any modern web browser (Chrome, Firefox, Safari, Edge).
3. That's it!

## 🌐 How to Deploy (GitHub Pages)
This repository is 100% ready to be deployed for free on GitHub Pages.
1. Upload/Push these files to a public GitHub repository.
2. In your repository on GitHub, negative to **Settings** > **Pages** (under the Code and automation section).
3. Under the *Build and deployment* header, set the **Source** to "Deploy from a branch".
4. Select the `main` or `master` branch and hit **Save**.
5. Wait a few moments, and GitHub will provide you with a live URL where your portfolio is hosted!
